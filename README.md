# Ralph Language for VS Code

## Overview
Ralph is the language used in alephium to create smart contracts, ralph-vscode is vscode extension.

## Features
- Highlighting ✅
- Snippet ✅
- Formatting ✅（simple format）
- IntelliSense ⏳
- Linting / Error Checking ⏳
- Debugging ⏳
- Deploy ⏳
